## 📝 Краткое описание программы

**Password Manager** — это консольное приложение для безопасного хранения, генерации и управления паролями. Программа позволяет:

- ✅ Добавлять новые записи (сервис, логин, пароль)
- ✅ Просматривать все сохраненные пароли
- ✅ Удалять записи по названию сервиса
- ✅ Генерировать случайные пароли с настраиваемыми параметрами:
  - Длина пароля
  - Использование строчных/заглавных букв
  - Использование цифр
  - Использование спецсимволов
- ✅ Автоматически сохранять данные в JSON файл
- ✅ Проверять корректность ввода перед сохранением

### 🏗 Архитектура
Приложение построено по паттерну **MVC** (Model-View-Controller) с соблюдением принципов ООП:
- **Инкапсуляция** — данные скрыты внутри классов
- **Наследование** — возможность расширения функционала
- **Полиморфизм** — гибкая работа с разными типами данных
